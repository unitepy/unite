## Unite.py is an open-source Python3 runtime environment.

### INSTALL:
```console
wget https://raw.github.com/unitepy/unite/main/unitepy.deb
sudo dpkg -i unitepy.deb
sudo apt install -f -y
```

### UNITE:
```console
#!/usr/bin/env unite
unite main.py
```

### UPM:
```console
upm init
upm install <package>
```
